## Setup

### Requirements
* [Node](https://nodejs.org/en)

### Local development
1. Create and fill environment file:
   ```shell
   cp .env.dist .env
   ```
2. Continue with [Global README.md](../../README.md)

#### If you want to run only Client app:
1. Install dependencies:
   ```shell
   npm install
   ```
2. Start application:
   ```shell
   npm run dev
   ```
